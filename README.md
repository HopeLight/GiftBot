使用说明:

1. 想完整的使用必须要McBotGift 搭配 MiraiBotGift 两者都是相辅相成

2. McBotGit -> 理论可以支持在Bukkit全版本 但是由于JDK的限制对于语法上导致低版本会出现问题,因为jdk 14 采用Switch 语法

   

本次测试环境:

> McBotGift -> Spigot 1.16.5
>
> MiraiBotGift -> localhost
>
> Mysql -> localhost
